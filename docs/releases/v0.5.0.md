# v0.5.0 — Dense automated workbench

Public release date: **2026-07-18**

This release makes the product denser for real client delivery: intake expansion,
local evidence verification, HTML dashboard, remediation SLAs, batch runs,
unknown-field warnings, denser auto packs, and deterministic `--as-of`.

## Customer commands

```bash
# Full automated run (default evidence verify on)
healthai-audit run inventory.json --out reports/client

# Minimal intake → expand → run
healthai-audit run samples/sample_intake_minimal.json --from-intake --out reports/riverside

# Or two-step
healthai-audit intake samples/sample_intake_minimal.json --out inventories/riverside.json --run

# Batch a folder of inventories
healthai-audit batch inventories/ --out reports/batch-2026-07-18 --as-of 2026-07-18

# Deterministic CI date
healthai-audit run inventory.json --as-of 2026-07-18 --out reports/ci
```

## What a run now writes

| Artifact | Purpose |
| --- | --- |
| `RUN_SUMMARY.md` | Operator front door |
| `dashboard.html` | Visual portfolio + tools + action queue |
| `owner-decision-packet.md` | Owner/MSP packet |
| `remediation-plan.md` | Owners + due dates by SLA |
| `report.json` | Full sanitized machine report |
| `kit-bridge/*` | Small Practice Security Kit import |
| `warnings.txt` | Unknown fields / missing high-impact fields |
| `vendor-followups.md` | Non-PHI vendor questions |
| `action-queue.csv` | Handoff rows with due dates |

## Dense features

### 1. Minimal intake expansion
Short card (practice + type + tool names) expands into a full inventory with
pack-aware seeds (scribe, imaging, scheduling, billing, RAG, MCP, therapy).

### 2. Local evidence verification
Optional (on by default for `run`): checks evidence `path` exists under the
inventory directory and optional `sha256` via binary hash only. Never reads
files as text. Rule `HA-EVID-003` when missing/mismatch.

### 3. Remediation plan
SLA defaults: block 14d, restrict 30d, conditional 45d, approve 90d.
Override via `remediation_defaults` or per-tool `owner` / `remediation_due`.

### 4. Denser auto packs
Additional `HA-PACK-*` rules for patient-facing escalation, PHI SaaS security
contact, dental scheduling approval, BH training posture, multi-state Rx,
MSP MCP allowlist evidence.

### 5. Schema warnings
Unknown inventory keys and missing high-impact fields surface in
`warnings` / `warnings.txt` without failing the run.

### 6. Deterministic `--as-of`
Locks review date, generated stamp, evidence expiry comparison, and
remediation due dates for CI/replay.

### 7. Batch mode
`healthai-audit batch dir/ --out out/` processes every `*.json` and writes
`BATCH_INDEX.md`.
